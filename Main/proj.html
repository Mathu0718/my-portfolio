Food Ordering Website

An interactive food ordering website that allows users to select a shop, view the menu, add items to the cart, and proceed to checkout.

Project Details

Users can explore menus from popular shops like McDonald's, Domino's, KFC, and Starbucks. A dynamic cart system ensures a smooth shopping experience. Items can be added or removed from the cart, with real-time price updates. Styled using CSS for an engaging and visually appealing interface. JavaScript enhances interactivity, handling cart operations and checkout. Fully responsive design for seamless use on different devices.

Technologies Used

HTML – Defines the structure and layout of the website.
CSS – Enhances styling and responsiveness.
JavaScript – Manages cart functionality and user interactions.

Tools Used

Code Editor – VS Code / Sublime Text
Browser Developer Tools – For debugging and testing
Image Editing Software – Optimizing food item images

Portfolio Website

A personal portfolio website showcasing projects, skills, and achievements. Designed with a clean and modern interface for an engaging user experience. Includes interactive sections such as About Me, Projects, Contact, and Resume. Responsive design ensures compatibility across different devices. Smooth animations and transitions enhance navigation and usability. Built to provide an online presence and highlight technical expertise.

Challenges Faced

I was inspired by many portfolio designs and gathered ideas from various sources. However, implementing the design and functionality took more time than expected. Balancing aesthetics with performance optimization was a key challenge. Ensuring smooth animations without affecting load time required careful adjustments. Debugging responsiveness issues across multiple devices was a learning experience. Through persistence, I refined the layout and created a fully functional, interactive portfolio.

Technologies Used

HTML – Defines the structure and content layout.
CSS – Enhances design, styling, and responsiveness.
JavaScript – Adds interactivity and smooth animations.

Tools Used

Code Editor – VS Code / Sublime Text
Browser Developer Tools – Debugging and performance optimization
Design Tools – Canva / Figma for UI enhancements
